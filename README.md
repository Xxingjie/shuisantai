# 水精灵变身啦

面向幼儿园中班幼儿的水的三种形态互动科学网页游戏。

## GitHub Pages 部署

1. 在 GitHub 新建一个仓库。
2. 将本文件夹内的全部文件上传到仓库根目录，不要只上传 `index.html`。
3. 打开仓库的 `Settings` → `Pages`。
4. 在 `Build and deployment` 中选择 `Deploy from a branch`。
5. 分支选择 `main`，目录选择 `/ (root)`，然后保存。
6. 等待 GitHub Pages 发布完成，访问页面显示的网址。

## 文件说明

- `index.html`：游戏入口。
- `assets/`：角色、冰块、标题及背景图片。
- `audio/`：14段中文配音。
- `.nojekyll`：关闭不需要的 Jekyll 处理。

本项目为纯静态网页，不需要服务器、数据库或构建命令。
